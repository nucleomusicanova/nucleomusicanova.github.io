Projetos

Bem-vindo ao repositório de projetos do Núcleo Música Nova. Aqui você encontrará projetos desenvolvidos por membros do Núcleo Música Nova. Principalmente aqueles que envolvem a criação de patches, programas, entre outros.

Ainda em Construção!

Synthia

Synthia é um conjunto de patches pensados para um contexto criativo-educacional.

Harry Partch Diamond

Esse projeto é uma breve explicação e exposição do funcionamento da Tonalidade Diamante de Harry Partch.

BiA

BiA é uma Biblioteca de Acústica, isto é, uma coleção de patches para explorar diversos tópicos de acústica musical e eletroacústica. Para usar os patches, utilize o software Pure Data disponível em https://puredata.info/.

Spectral Tools

Material anexo para o texto “Espectralismo: um guia composicional” de Felipe de Almeida Ribeiro.
